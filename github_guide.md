# 🚀 GitHub Pages-ке сайтты салу нұсқаулығы

## Не қажет?
- GitHub аккаунты (тегін): https://github.com
- `chemistry_quiz.html` файлы (дайын ✅)

---

## 📋 Қадамдар (5 минут)

### 1-қадам — GitHub-та тіркел
Егер аккаунтың жоқ болса:
1. https://github.com сайтына кір
2. **"Sign up"** басып, email + пароль енгіз
3. Тіркелуді аяқта

---

### 2-қадам — Жаңа репозиторий жаса
1. GitHub-та жоғарғы оң жақтағы **"+"** белгісін басып → **"New repository"**
2. **Repository name** өрісіне жаз:
   ```
   chemistry-quiz
   ```
3. **"Public"** деп таңда (міндетті!)
4. **"Add a README file"** чекбоксын белгіле
5. **"Create repository"** батырмасын бас

---

### 3-қадам — Файлды жүктеп сал
1. Жаңа репозиторий бетінде **"Add file"** → **"Upload files"** бас
2. `chemistry_quiz.html` файлыңды сүйреп апар немесе **"choose your files"** арқылы тап
3. **Маңызды:** файл атын өзгерт → **`index.html`** деп атауың керек!
   > (Жүктегенде файл атын `index.html` деп өзгерт — GitHub Pages негізгі бет ретінде `index.html`-ді іздейді)
4. Төменде **"Commit changes"** батырмасын бас

---

### 4-қадам — GitHub Pages қосу
1. Репозиторийдің жоғарғы мәзірінде **"Settings"** бас
2. Сол жақ тізімде **"Pages"** тап
3. **"Branch"** бөлімінде:
   - `None` → **`main`** деп өзгерт
   - Қалтаны `/root` деп қалдыр
4. **"Save"** басс

---

### 5-қадам — Сілтемені ал
Бірнеше минут күт (1-3 минут), содан соң:

```
https://СЕНІҢ_АТЫҢ.github.io/chemistry-quiz/
```

Мысалы егер атың `altynbek2026` болса:
```
https://altynbek2026.github.io/chemistry-quiz/
```

Осы сілтемені досыңа жіберсең болды! 🎉
